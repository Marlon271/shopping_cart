# HU-107 Tablero Operativo

**Como** usuario del sistema  
**Quiero** gestionar la canasta desde una interfaz visual unificada  
**Para** operar el flujo completo sin depender de Postman

## Criterios de aceptacion

- el tablero permite abrir o reutilizar una canasta
- el tablero permite registrar, ajustar y retirar lineas
- el resumen lateral muestra el consolidado monetario
- la experiencia se adapta a escritorio y pantallas angostas

## Artefactos asociados

- frontend `Vue 3`
- store `basketStudioStore`
- evidencia visual en `Doc/Evidencia`
- rama de trabajo `HU-107-basket-dev`

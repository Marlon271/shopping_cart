# Priorizacion MoSCoW

## Must Have

- abrir o reutilizar una canasta activa
- registrar lineas de producto
- consultar detalle de la canasta
- ajustar unidades
- retirar lineas
- calcular montos globales
- tablero web funcional

## Should Have

- interfaz responsiva para escritorio y pantallas angostas
- persistencia en PostgreSQL con restricciones e indices
- documentacion navegable por historias de usuario

## Could Have

- atajos para plantillas de productos frecuentes
- filtros por categoria o segmento de compra
- exportacion del consolidado

## Wont Have por ahora

- proceso de checkout
- autenticacion formal
- catalogo externo de productos
